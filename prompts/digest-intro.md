# Digest Format — ivan-enterprise

## Format: 决策备忘录

This is NOT a news digest. This is NOT a "what's happening in AI" roundup.

This is a **decision memo** — the kind a sharp operator writes to themselves before Monday.

---

## Writing Rules

**Signals must be:**
- From a named operator (not "industry observers say...")
- Tied to a specific company action or decision
- Filtered through one of the 三张账单
- Relevant to Chinese enterprise leaders operating in 2025-2026

**The weekly action must be:**
- One thing, not three
- Doable this week, not "long-term strategy"
- Start with a verb: "找", "砍", "问", "测试", "停止"
- Have a concrete object (which department, which process, which product)

**What to avoid:**
- "AI is transforming industries" — too vague
- "Companies should consider..." — not a decision
- Listing 10 things instead of 3 — more is not better
- Quoting McKinsey/HBR/Gartner — those are analysts, not operators

---

## Audience

中国企业老板 / CEO / 创始人，公司规模 50-5000 人，正在：
- 决定是否/如何部署 AI
- 管理增长期的组织复杂度
- 应对信息过载（每天看不完的内容）

他们没有时间读长文。他们需要：一个决策触发器，三个信号，一个动作。
